Sobre o Livro
Descrição

Programação orientada a objetos com JAVA é o primeiro livro introdutório que integra completamente o BlueJ ao ensino da orientação a objeto utilizando JAVA, permitindo ao aluno criar objetos de qualquer classe e interagir com seus métodos.

Em seus 13 capítulos, ele apresenta tópicos desafiadores que possibilitam o aprofundamento do aprendizado, problemas analisados antes de serem introduzidas construções de linguagem e referências que direcionam o aluno para outras leituras e fontes suplementares.

Completo e didático, Programação orientada a objetos com JAVA é voltado para os cursos de ciência da computação e análise de sistemas, mas é indicado também para profissionais de desenvolvimento.
